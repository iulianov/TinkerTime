TinkerTime

Automatically Updating Mod Manager for Kerbal Space Program

View project on GitHub

Features

  • Platform Agnostic (Run on Windows, OSX, Linux)
  • Automatically keep your mods up-to-date from your favourite mod hosts:
    • KerbalStuff
    • Github
    • Curse
  • Keep Module Manager up-to-date
  • You can still add mods as zip files
  • Know when mod updates are available
    • Know if your mod supports the latest KSP Version
  • Keeps itself up-to-date
  • Import/Export Mod Lists
